# GameShop Static Ecommerce Demo

A static web storefront built with HTML, CSS, and vanilla JavaScript.

## Features

- Product browsing with search and genre filter
- Login and registration using localStorage
- Shopping cart with quantity controls
- Order checkout and order history
- Admin dashboard for managing games, users, and orders
- Admin account: `admin@example.com` / `admin123`
- Sample customer account: `user@example.com` / `user123`

## How to run

1. Open `index.html` in your browser.
2. Use the navigation buttons to view products, cart, and orders.
3. Login or register to enable cart and order functionality.
4. Login as admin to access the dashboard.
